피보나치 수열
$F_n$ = 0(if n=0)
	= 1(if n=1)
	= $F_{n-1} + F_{n-2}$

```
RecFIBO(n):
	if n=0
		return 0
	else if n=1
		return 1
	else 
		return RecFIBO(n-1) + RecFIBO(n-2)
```

T(n) = T(n-1) + T(n-2) + O(1)
T(n) > $F_n$

```
MEMFIBO(n):
	if n=0
		return 0
	else if n=1
		return 1
	else 
		if F[n] is undefined
			F[n] = MEMFIBO(n-1) + MEMFIBO(n-2)
		return F[n]
```

```
ITERFIBO(n): //해당 값까지의 피보나치 수열
	F[0] = 0
	F[1] = 1
	for 2..n
		F[i] = F[i-1] + F[i-2]
	return F[n]
```
```
ITERFIBO2(n): //해당 인덱스 값 찾기
	prev = 0
	curr = 1
	for 1..n
		next = curr + prev
		prev = curr
		curr = next
	return curr
```
-> O(n)

Subset Sum
```
//resusive algorithm -> O(2^n)
SUBSETSUM(X, i, T): //X에서 i까지
	if T=0
		return True
	else if T<0 or i=0
		return False
	else
		with = SUBSETSUM(X, i-1, T-X[i])//X[i]를 포함
		wout = SUBSETSUM(X, i-1, T)//X[i]를 버림
		return with or wout
```
1 ≤ i ≤ n+1 and 0 ≤ t ≤ T
2차배열S[1..n+1, 0..T]를 써서S[i, t]에 SS(i, t)를 기억
i는 n일 때 base case -> 감소하는 방향
t는 0일 때 base case -> 증가하는 방향
```
FastSubsetSum(X[1..n], T):
	S[n+1, 0] = True
	for t=1..T
		S[n+1, t] = False
	for i=n..1

		for t=1..X[i]-1 //X[i]보다 작은 값
			S[i,t] = S[i+1, t] //행렬에서 오른쪽 값
		for t=X[i]..T

//O(nT)
```
